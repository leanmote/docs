# REST API

The REST API is the primary integration surface for Leanmote platform entities.

Use it when you need programmatic access to organizations, teams, users, integrations, and dashboard resources.

## Best use cases

- Sync Leanmote metadata into internal data platforms.
- Build custom reporting workflows.
- Extend internal tooling with Leanmote context.

## Core resource families

- Organizations
- Teams
- Users
- Git and integrations
- Dashboard views

## Authentication and requests

REST endpoints use bearer token authentication.
