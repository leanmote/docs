# SCIM Handler

SCIM support enables automated user lifecycle management for enterprise organizations.

Use SCIM when you need identity provisioning and deprovisioning aligned with your corporate identity provider.

## What SCIM improves

- Faster user onboarding/offboarding
- Reduced manual access management effort
- Better consistency for user/team mapping integrity

## Typical enterprise setup

1. Configure SSO with your identity provider.
2. Enable SCIM provisioning for Leanmote.
3. Map groups to organizational access patterns.
4. Validate deprovisioning and role transitions.
