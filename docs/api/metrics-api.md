# Metrics API

The Metrics API is optimized for analytical payloads and metric-specific computations.

Use it when you need detailed delivery, flow, collaboration, and financial metric outputs for automation or BI pipelines.

## Best use cases

- Scheduled extraction of engineering KPIs
- Advanced trend and variance analysis in external tooling
- Team and portfolio analytics workflows

## Metric families

- DORA metrics
- Flow metrics
- Collaboration metrics
- Meeting and interaction metrics
- Project financial metrics
- AI impact metrics

For the complete route list, see [METRICS-TEST Endpoints](metrics-test.md).
