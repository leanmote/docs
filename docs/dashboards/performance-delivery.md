# Performance Delivery

Performance Delivery is the operational core dashboard for engineering teams.

It is designed to answer: are we shipping predictably, quickly, and with acceptable quality?

## Main tabs and what they show

- **DORA**: deployment frequency, lead time, change failure rate, and recovery indicators.
- **Flow**: throughput, cycle time, work in progress, and waiting-time behavior.
- **Collaboration**: review and interaction signals that influence delivery velocity.

## Typical filters

- Date range (weekly, sprint, monthly)
- Team/squad
- Repository and project scope
- Workflow segment (when available)

## How to run the review

1. Start with DORA trend direction.
2. Validate flow bottlenecks (cycle and WIP pressure).
3. Check collaboration metrics for review latency.
4. Capture one improvement action per team for the next cycle.

## Best for

- Engineering managers
- Team leads
- Operational delivery ceremonies
