# Dashboards Overview

Dashboards are where Leanmote turns integrated operational data into decisions.

Each dashboard supports a different layer of engineering management, from team execution to portfolio strategy.

## Dashboard suite

- [Performance Delivery](performance-delivery.md): day-to-day delivery quality and speed.
- [Project Financials](project-financials.md): output and investment signals by team and scope.
- [Strategic Overview](strategic-overview.md): executive trend alignment and portfolio direction.
- [Bottleneck Insights](bottleneck-insights.md): friction detection across workflow stages.
- [AI Tools](ai-tools.md): impact and adoption signals for AI-assisted delivery.

## How teams use dashboards effectively

- Managers run weekly execution reviews.
- Directors run cross-team trend and capacity reviews.
- Executives run monthly strategic and investment reviews.

## Shared filtering model

Most dashboard views support filtering by:

- Date range
- Team
- Repository/project/board scope
- Workflow or metric dimensions

Keeping filter definitions standardized improves comparability across teams.
